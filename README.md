# FloraMind 🌱

## Description
FloraMind is a plant disease detection system that identifies diseases from plant leaf images using machine learning.

## Features
- Upload plant leaf image
- Detect plant disease
- Display results instantly

## Technologies Used
- Node.js
- Express.js
- Python (for ML)
- TensorFlow

## How to Run

```bash
# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm start
```

## Environment Variables

Create a `.env` file inside the backend folder and add:

```env
PORT=5000
API_KEY=your_api_key_here
```
